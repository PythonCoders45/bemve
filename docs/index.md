# Bemve Animation Engine Documentation

Welcome to the official documentation for **Bemve**—a modular, Python-native animation framework for technical, mathematical, and data-driven video creation.

---

## Key Features

- **Vector Precision (`VMobject`)**: Cubic Bézier paths for shapes, path tracing (`Create`), and morphing (`Transform`).
- **Data & 3D Visualizations**: Native Matplotlib & Seaborn integration for animated growing charts and rotating 3D surface graphs.
- **Canva-Style FX**: Built-in transitions like `FadeIn`, `SlideIn`, and `GrowFromCenter`.
- **Media Support**: Load `.ttf` custom fonts, `.png` images, and stream `.mp4` video clips into vector scenes.
- **Smart Build Pipeline**: MD5 hash-based partial clip caching and automatic browser preview.
